    因为目前项目中可以使用es6代替传统的es5代码，所以这边总结一下项目中常用的es6代码

    # let和const

    定义变量使用let和const，常量使用const，基础变量建议使用let定义。

    # 箭头函数（=>）

    定义函数建议使用箭头函数，使用清晰，不过需要注意里面this的指向问题，箭头函数里面的this在定义的时候就已经规定了，不是传统的调用的时候才出现的环境。

    # 对象的函数变量缩写

    例如ajax成功回调之前是使用success：function(result){},现在可以使用success(result){}代替

    # 数组的遍历

    以前我们常用的基础循环遍历数组，获取元素等等，或者使用foreach遍历元素，当然foreach存在一个缺陷，不能使用break或者return中断或者回到外层函数。现在我们可以使用forof循环对数组进行循环。for(let value of arr){}。当然forof还可以遍历set，map集合。

    # 对象的遍历

    遍历对象我们可以使用es6带的Object.values。es6新带了Object.values和Object.keys,这两个函数处理是返回处理对象的一个数组，key返回的是对象的键数组，values返回的是处理对象的值数组。同时里面可以return false进行中断

    # 解构的使用

    交换两个值 [a,b] = [b,a]。提取接口返回的值并进行处理，

    let jsonData = {
  id: 42,
  status: "OK",
  data: [867, 5309]
};

let { id, status, data: number } = jsonData;
//data:number 其实number是定义的变量，data是实际匹配的数据

    # 模板字符串

    这边我并不很推荐使用es6的模板字符串，使用目前的脚本方式，待用mustache渲染应该也可行。

    # 函数默认值

    函数传参的时候可以带默认值 function test(key = 1){}

    # rest参数

    可以使用...代替所有函数参数

    # Array.from()

    该函数可以将类数组转化为真实的数组，最常见的就是获取到的dom数组转为真实的数组，并进行循环等操作。

    # Array.every()

    该方法，用于遍历数组的每一个元素，看是不是数组内的每一个值都满足该方法，满足的话返回true，有一个不满足则返回false

    var ages = [32, 33, 16, 40];

function checkAdult(age) {
    return age >= 18;
}

function myFunction() {
    document.getElementById("demo").innerHTML = ages.every(checkAdult);
}

    # array.filter()

    用于对数组中每个元素进行过滤，返回的输一个新数组，里面的值为满足filter里面条件的所有元素

    const filtered = [1, 2, 3].filter(element => element > 1);
// filtered: [2, 3];
