# 垂直居中（重点）

1.行内元素 （单行文字垂直居中）

设置line-height = height

2.块级元素，当然这边只是列举出了几个常用的情况，其实还有很多其他的解决方案

<div class="parent">
   <div class="child">child</div>
</div>
1

2.1 使用定位的方式，直接使用定位，将子元素的距离父元素的位置设置为50%并偏移高度的一半

.parent {
    position: relative;
    height: 200px;
}
.child {
    width: 80px;
    height: 40px;
    background: blue;
    position: absolute;
    left: 50%;
    top: 50%;
    margin-top: -20px;
    margin-left: -40px;
}

2.2 使用定位+transform，不需要知道子元素的尺寸

.parent {
    position: relative;
    height: 200px;
}
.child {
    width: 80px;
    height: 40px;
    position: absolute;
    left: 50%;
    top: 50%;
    transform: translate(-50%, -50%);
    background: blue;
}

2.3 使用绝对定位 + margin:auto 这个方法以前还没有接触过，这边学习到了。

.parent {
    position: relative;
    height: 200px;
}
.child {
    width: 80px;
    height: 40px;
    position: absolute;
    left: 0;
    top: 0;
    right: 0;
    bottom: 0;
    margin: auto;
    background: blue;
}

具体这边为什么这样子设置就可以居中，简单看了下介绍，可以这么理解。我们设置left，right这种对立属性的时候，元素就会展示出他的流体性，如果我们不设置宽高，宽度就会充满父级，但是我们此时设置了高度，那么设置margin：auto属性就会自动平分剩下的高度，可以这么理解的吧。

    width: 600px;

    border: 1px solid red;
    display: table;
}
.child {
    display: table-cell;
    vertical-align: middle;

2.6 calc()

.parent {
    width: 300px;
    height: 300px;
    border: 1px solid red;
    position: relative;
}
.child {
    width: 100px;
    height: 100px;
    background: blue;
    padding: -webkit-calc((100% - 100px) / 2);
    padding: -moz-calc((100% - 100px) / 2);
    padding: -ms-calc((100% - 100px) / 2);
    padding: calc((100% - 100px) / 2);
    background-clip: content-box;
}
